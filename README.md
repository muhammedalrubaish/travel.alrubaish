# رحلة كيب تاون والجاردن روت — يوليو 2026 🇿🇦

مخطط رحلة تفاعلي باللغة العربية (RTL) لشخصين إلى كيب تاون والجاردن روت، في ملف HTML واحد بدون أي اعتماديات بناء.

## التشغيل
افتح `index.html` مباشرة في المتصفح — لا يحتاج خادماً أو تثبيت حزم.

## المزايا
- **واجهة داكنة عصرية** بألوان `#222831` / `#FF5722` / `#76ABAE` وخط Tajawal.
- **خريطة مسار SVG تفاعلية**: الضغط على أي محطة ينقلك بسلاسة إلى يومها، وتتحدث تلقائياً مع تعديل الأيام.
- **محاكاة الرحلة**: سيارة متحركة تقطع المسار محطةً محطة مع عرض تفاصيل كل يوم.
- **السكن في كيب تاون والجاردن روت (نيسنا) فقط** — هيرمانوس وبقية المحطات زيارات على المسار بدون مبيت.
- **تعديل كامل للجدول**: إضافة/حذف/تعديل الأيام والأنشطة والروابط، واختيار موقع كل يوم من قائمة أو بالنقر على الخريطة، مع متحكم بعدد أيام الرحلة.
- **قائمة مهام السفر (To-Do)**: مهام جاهزة مقسّمة (مستندات وحجوزات · قبل السفر · الأمتعة · خلال الرحلة) مع خيار "تم ✓" يشطب المهمة، شريط تقدّم بنسبة الإنجاز، وإضافة/حذف مهام خاصة — كلها تُحفظ تلقائياً.
- **صور تلقائية للأماكن والفعاليات** عبر Wikipedia API مع إمكانية تغييرها، وروابط رسمية للمواقع والفعاليات في كل يوم.
- **حاسبة ميزانية مرنة**: مسافرون، أسعار قابلة للتعديل، اختيار فنادق مع وضع تعديل (اسم/سعر/صورة)، وتوزيع ليالٍ تلقائي بين المدينتين.
- **طرق الدفع**: كاش (خصم 5% على الفنادق والطيران) أو تابي/تمارا — **4 أقساط على الطيران والفنادق فقط** والباقي يُدفع مباشرة.
- **أزرار حجز مباشرة**: Booking.com وAirbnb وAgoda وExpedia بتواريخ وأعداد المسافرين الفعلية، وروابط طيران Google Flights وSkyscanner.
- **تحويل عملات مزدوج فوري (SAR ⇄ ZAR)** بجانب كل بند، مع محوّل يدوي وسعر صرف قابل للتعديل.
- **دليل يوليو**: طقس كل منطقة، مقابس الكهرباء (Type M/N · 230V)، نصائح الأمان وأرقام الطوارئ.
- **طباعة / PDF**: تنسيق ورقي بخلفية بيضاء مع إخفاء عناصر التحكم تلقائياً.
- **حفظ تلقائي (LocalStorage)**: كل التعديلات تُحفظ وتُستعاد تلقائياً، مع زر استعادة الافتراضي.
