# رحلة كيب تاون والجاردن روت — يوليو 2026 🇿🇦

مخطط رحلة تفاعلي باللغة العربية (RTL) لشخصين إلى كيب تاون والجاردن روت، في ملف HTML واحد بدون أي اعتماديات بناء.

## التشغيل
افتح `index.html` مباشرة في المتصفح — لا يحتاج خادماً أو تثبيت حزم.

## المزايا
- **واجهة داكنة عصرية** بألوان `#222831` / `#FF5722` / `#76ABAE` وخط Tajawal.
- **خريطة مسار SVG تفاعلية**: الضغط على أي محطة ينقلك بسلاسة إلى يومها في الجدول مع تمييز بصري.
- **جدول رحلة لـ 11 يوماً / 10 ليالٍ**: كيب تاون (5) → هيرمانوس (1) → نيسنا (3) → أودتشورن (1) مع نصيحة عملية لكل يوم.
- **حاسبة ميزانية مرنة**: عدد المسافرين، أسعار الطيران والمواصلات والأنشطة والطعام قابلة للتعديل المباشر، واختيار الفنادق من قوائم منسدلة.
- **وضع تعديل الرحلة**: تغيير أسماء الفنادق وأسعارها وصورها (روابط URL) وحفظها.
- **طرق الدفع**: كاش (خصم 5% على الفنادق والطيران) أو تقسيط تابي/تمارا (4 أقساط متساوية).
- **تحويل عملات مزدوج فوري (SAR ⇄ ZAR)** بجانب كل بند وفي الإجمالي، مع محوّل يدوي وسعر صرف قابل للتعديل.
- **دليل يوليو**: طقس كل منطقة، مقابس الكهرباء (Type M/N · 230V)، نصائح الأمان وأرقام الطوارئ.
- **طباعة / PDF**: تنسيق ورقي بخلفية بيضاء مع إخفاء عناصر التحكم تلقائياً.
- **حفظ تلقائي (LocalStorage)**: كل التعديلات تُحفظ في المتصفح وتُستعاد عند العودة، مع زر استعادة الافتراضي.
